#Gulp install
----------
**安装nodejs -> 全局安装gulp -> 项目安装gulp以及gulp插件 -> 配置gulpfile.js -> 运行任务**

##install

1.  全局安装 gulp：
>全局安装gulp目的是为了通过她执行gulp任务；   
>查看是否正确安装：命令提示符执行gulp -v，出现版本号即为正确安装。   

```
    $ npm install --global gulp
```

2. 作为项目的开发依赖（devDependencies）安装：

```
    $ npm install --save-dev gulp
```
